# ClickAssist Responsible Use

Last updated: April 21, 2026

ClickAssist is a user-controlled automation tool.

Use ClickAssist only in apps, games, services, and workflows where automation is permitted.

Do not use ClickAssist for:

- Spam
- Fraud
- Fake engagement
- Harassment
- Deceptive behavior
- Abuse of third-party services

The developer is not responsible for misuse by end users.
