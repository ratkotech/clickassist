# Features

## Run Sources

ClickAssist separates automation into two source types.

- Click Points: manually captured on-screen targets.
- Mimic Pattern: recorded taps and swipes converted into a reusable pattern.

Presets are source-scoped. A Click Points preset restores click points. A Mimic preset restores mimic data.

## Click Points

- Capture real on-screen targets with the point picker overlay.
- Edit or remove saved targets.
- Use multi-click mode for multiple steps.
- Choose sequential or simultaneous execution.

## Mimic Patterns

- Record natural taps and swipes in the mimic recorder.
- Classify gestures into tap or swipe actions.
- Edit delays, press duration, action type, start target, and swipe target.
- Clear mimic patterns when needed.

## Timing Controls

- Speed presets: Slow, Normal, Fast, Turbo, Ultra.
- Custom interval from 10 ms to 5000 ms.
- Start delay presets and custom delay.
- Count mode or infinite mode.

## Presets

- Save current setup with a custom name.
- Load, edit, and delete presets.
- Import presets from JSON.
- Export presets as JSON from Settings / Data Actions.

## Overlay Controls

- Floating controls for starting and stopping automation from other apps.
- Compact native control bar.
- Overlay visibility depends on permissions and app lifecycle.

## Safety UX

- Permission disclosures before sensitive Android settings.
- Help & Safety section.
- Responsible-use page.
- Local privacy summary.
- Device-health checks for battery level, thermal state, run intensity, and setup readiness.
