# ClickAssist Privacy Policy

Last updated: April 27, 2026

## Overview

ClickAssist is an open-source Android automation tool that lets users configure tap targets, swipe routes, presets, and overlay controls on their own devices.

## What ClickAssist Stores

ClickAssist stores the following data locally on your device:

- Saved presets
- Click points and pattern steps
- Mimic pattern steps
- Onboarding completion state
- Visual and runtime preferences related to automation setup

This local storage is used only so the app can remember your configuration and show the current setup state the next time you open it.

## What ClickAssist Does Not Collect

Based on the current open-source project codebase:

- No cloud sync is included
- No analytics SDK is included
- No crash-reporting SDK is included
- No account system is included

## Permissions and Sensitive Access

ClickAssist may ask you to enable:

- AccessibilityService, to simulate taps and swipes you configure
- Display over other apps, to show floating controls and the point picker
- Notifications, to show foreground status and quick controls
- Battery optimization exemption, if you choose to improve background reliability

These permissions are user-controlled. ClickAssist does not enable them automatically.

## Third-Party Rules

You are responsible for using ClickAssist only where automation is permitted. The app does not verify whether third-party apps, games, or services allow automation.
